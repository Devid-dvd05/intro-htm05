Conhecendo o elemento de imagem

Sobre imagens

Trabalhar com imagens no HTML envolve muito mais do que apenas colocar uma foto na página.

Existem várias formas tecnicas e recursos para exibir, organizar, otimizar e adaptar imagens para diferentes dispositivos.

Formatos mais comuns de imagens

  • JPEG/JPG: Fotos e imagens compactadas
  • PNG: Suporta transparência
  • WEBp: Formato moderno e leve (também suporta transparência)
  • Gif: Animações simples
  • Avif: Excelente qualidade com um tamanho pequeno

Sobre o elemento img

A forma mais comum de inserir imagens no HTML, é usando a tag img

Exemplo pratico

Trabalhando com links externos/URL de sites

SRC (source/fonte)- Atributo responsável pelo caminho da imagem, ou seja, defini onde está a imagem.

Trabalhando com imagens locais

ALT (alternative/alternativo) - É o elemento que será apresentado caso a imagem não carregue.

O atributo alt é MUITO IMPORTANTE

  • Acesseibilidade
  • Leitores de tela
  • SEO
  • Fallback

Ajustando o Tamanho das Imagens

HEIGTH - O atributo height serve para ajustar o tamanho da altuta da imagem

width - O atributo width serve para ajustar o tamanho da largura da imagem
